# How To Draw 


## Installation

- visit mongodb site and take the steps to install mongodb driver [here](https://docs.mongodb.com/manual/installation/).
- Install Node.js from [here](https://nodejs.org/en/download/).
- inside folder how-to-fraw-frontend run 
```bash
npm install
npm start
```
- inside folder how-to-fraw-backend run 
```bash
npm install
cd server
node server.js
```
- you can test endpoints through https://localhost:3000/ 
- you can test front-end through https://localhost:3000/
